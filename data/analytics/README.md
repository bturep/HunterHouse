# Analytics rollups

Permanent monthly snapshots of site usage, kept here because Cloudflare Analytics
Engine only retains raw data points for ~90 days. These files outlive that window.

- **`<YYYY-MM>.json`** — one file per month: totals, top items, top searches
  (including zero-result searches), downloads by collection, and coarse country
  counts.
- **`monthly-summary.csv`** — one row per month (rebuilt from the JSON files) for
  an at-a-glance trend.

Generated by `scripts/rollup_analytics.py`, run automatically on the 1st of each
month by `.github/workflows/rollup-analytics.yml` (or manually from the Actions
tab). **Aggregate only** — no cookies, no IP addresses, no per-visitor data.

Note: the current month's `<YYYY-MM>.json` is partial until the month closes; the
1st-of-next-month run rewrites it complete.
